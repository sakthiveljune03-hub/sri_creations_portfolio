# Sri Creations Portfolio

This repository contains the complete codebase for the Sri Creations Portfolio project.

## Repository Structure

The project is structured as a monorepo:

- **[frontend/](./frontend)**: Frontend React web application built with Vite and TailwindCSS.
- **[backend/](./backend)**: Backend Node.js/Express API with database configuration and services.
