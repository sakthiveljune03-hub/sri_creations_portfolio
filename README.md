# Sriman Cinematic Portfolio — Frontend

A premium, single-page, highly immersive 3D parallax and glassmorphic portfolio web application designed for a professional cinematic video editor and colorist.

---

## 🚀 Key Features

*   **Interactive 537-Frame Scroll Sequence**: High-performance canvas-based scroll playback driven by user scrolling state.
*   **Cinema Mode**: Seamless backdrop dimming to focus on showcase reels and media playback.
*   **3D Parallax Hero**: Parallax layout utilizing Google Fonts, Branche display font, and customized web assets.
*   **Glassmorphic Design System**: Custom `.white-glass` variables, blurs, glow shadows, and interactive components.
*   **Responsive Fluid Grid**: Optimized for Mobile, Tablet, Laptop, and Desktop screens.
*   **Interactive Reviews**: Carousel for client testimonials and active scroll-offset navigations.
*   **PWA Ready**: Progressive Web App capabilities for offline fallbacks and local installations.

---

## 🛠️ Technology Stack

*   **Core**: React 19, TypeScript, HTML5
*   **Styling**: TailwindCSS (v4.0+), Custom Vanilla CSS Variables
*   **Animations**: GSAP, Motion
*   **3D Elements**: Three.js
*   **Icons**: Lucide React

## 💻 Getting Started

### Prerequisites

*   [Node.js](https://nodejs.org/) (v18.0 or higher recommended)
*   [npm](https://www.npmjs.com/)

### Installation

1.  Clone this repository and switch to the `frontend` branch:
    ```bash
    git clone -b frontend https://github.com/Anbu-ss/sri_creation-portfolio.git
    cd sri_creation-portfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Set up local environment variables:
    ```bash
    cp .env.example .env
    ```

4.  Start the development server:
    ```bash
    npm run dev
    ```

5.  Build the project for production:
    ```bash
    npm run build
    ```
